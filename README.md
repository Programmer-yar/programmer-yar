## Selected Projects

Ingests property data from multiple sources, runs valuation logic, and uses LLMs to generate market-ready output. Designed cost controls around expensive paid APIs so the tool stays economical to run day-to-day. Not public — abuse and API-cost risk. Walkthroughs on request.

### Business reports automation *(client work)*
End-to-end pipelines that pull data from platforms like Amazon Seller Central + Ads, transform it, and deliver scheduled reports into Slack and email. Rate-limit gymnastics, retries, per-client scoping, run monitoring. Runs unattended.

### Multi-agent competitor research
CrewAI-based agents that collapse an analyst-day of competitor research and report generation into ~15 minutes. Structured outputs, monitored runs, stage-by-stage traceability. Built at Andiron AI.

### Slack AI assistant with function calling
Conversational tool routing in Slack — user asks in natural language, ChatGPT function calling parses and validates the request, routes to the right internal tool, and returns results in-channel. Replaces "log into the dashboard" with "ask in Slack".

### Retrieval + vector search for keyword optimization
OpenAI embeddings over a large competitor keyword corpus, retrieval for the most relevant terms, then LLM-driven listing generation. One module of an 8-stage Amazon listing-optimization pipeline.
